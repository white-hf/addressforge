# AddressForge 业务专家版产品文档 (v1.0)

## 1. 产品愿景
将 AddressForge 从“ML 实验管理工具”重构为“地址业务专家工作台”，实现数据处理全流程的**可视化治理与闭环管控**。通过业务视角屏蔽底层 ML 细节，让非技术业务人员能够高效完成地址质量监控、错误纠偏与资产发布。

## 2. 核心功能布局 (业务全流程导航)
*   **概览中心 (Overview)**：数据全流程“水位”监控，实时掌握各阶段处理进度。
*   **地址治理 (Address Governance)**：查看清洗性能指标，错误桶分析。
*   **纠错中心 (Expert Review)**：侧边栏比对界面，执行人工校准与 Gold Label 沉淀。
*   **资产中心 (Assets Library)**：管理标准地址快照与版本发布。

## 3. 多语言策略 (I18N)
*   **术语标准化**：维护中英文映射词典。
*   **实时切换**：支持 UI 级别的一键中英文切换。
